# Deep Learning Projects using TensorFlow & Keras

🚀 This repository contains beginner-friendly deep learning projects implemented while learning and exploring neural network architectures using TensorFlow and Keras. The project demonstrates practical implementations of Artificial Neural Networks (ANN), Convolutional Neural Networks (CNN), Recurrent Neural Networks (RNN), and Long Short-Term Memory (LSTM) models for different prediction and classification tasks.

## 📌 Project Overview

The main objective of this repository is to understand the fundamentals of deep learning, neural network architectures, data preprocessing, model training, evaluation, and performance analysis through hands-on implementation.

This project includes:
- Artificial Neural Network (ANN)
- Convolutional Neural Network (CNN)
- Recurrent Neural Network (RNN)
- Long Short-Term Memory (LSTM)

## 🧠 Deep Learning Models Implemented

### 1️⃣ Artificial Neural Network (ANN)
- Built a fully connected neural network model.
- Applied preprocessing techniques and feature scaling.
- Used for prediction/classification tasks.
- Evaluated performance using accuracy and loss metrics.

### 2️⃣ Convolutional Neural Network (CNN)
- Implemented CNN architecture for image classification.
- Used convolutional and pooling layers for feature extraction.
- Improved model learning from image-based datasets.

### 3️⃣ Recurrent Neural Network (RNN)
- Implemented RNN model for sequential data processing.
- Learned temporal patterns and sequence dependencies.

### 4️⃣ Long Short-Term Memory (LSTM)
- Built an LSTM model for better sequence prediction.
- Handled long-term dependencies in sequential/time-series data.
